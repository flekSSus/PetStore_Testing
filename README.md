# Petstore API Testing

Автоматизированные тесты для REST API [Petstore](https://petstore3.swagger.io/) - учебного сервиса для управления питомцами.  
Проект демонстрирует навыки API-тестирования, создания коллекций Postman, написания автотестов (Pre‑request/Post‑response скрипты), параметризации.

## 📖 Документация API
[Swagger UI Petstore](https://petstore3.swagger.io/) — интерактивная документация с возможностью выполнять запросы.

## 🛠 Стек технологий
- **Postman** - создание и запуск коллекций, написание тестов
- **Newman** - запуск коллекций из командной строки
- **SwaggerUI / OpenAPI** - анализ документаци
- **Node.js** - для установки Newman
